## Node
### Node介绍
#### Node简介
Node.js是一个开源和跨平台的JavaScript运行时环境,Node基于V8。
```
V8引擎是一个JavaScript引擎，作用是将JavaScript翻译成操作系统CPU能够读懂的代码（将JS代码翻译成机器代码，然后交由CPU去执行）
```
#### Node优势
有一个独特的优势，因为数百万为浏览器编写JavaScript的前端开发人员现在可以在编写客户端代码的同时编写服务器端代码，而不需要学习一种完全不同的语言。

#### 退出Node程序
- 在控制台中可以通过ctrl-C
- 在代码中process.exit()

#### 读取环境变量
通过process.env
